# Trip Planner Agent
The Trip Planner Agent is designed to assist users in planning their trips by providing weather details for specific locations and dates. It can also save the trip plan details for future reference.

# Tools 
- **get_weather_details**: Get the weather details for a specific location and date.
- **save_my_trip_plan**: Save the trip plan details for future reference.

# Fetaures to add:
- **get_hotels**: Get hotel details for a specific location and date.
- **book_hotel**: Book a hotel for a specific location and date.
- **get_train_details**: Get train details for a specific location and date.
- **book_train**: Book a train for a specific location and date.
